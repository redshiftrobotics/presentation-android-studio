WiFi Direct

Allows for communications between the Drive Station and Robot Controller

Settings>Airplane mode>On

Settings>WiFi>On

Settings>WiFi>Overflow Settings (three dots)>WiFi Direct

WiFi Direct Configuration

Overflow Settings>Rename Device

[TEAM NUMBER]-[ds/rc]

i.e. 2856-rc and 2856-ds

Establish connection between devices

Developer Options

Adds functionality to the communications between the Android device and the computer

Needs to be enabled in order to upload apps and receive debugging information from the phone

Settings>About Phone

Tap 'SW Version' 7 times

Enabling USB Debugging

Settings>Developer Options>USB Debugging

Drivers (Windows Only)


The phone will connect as a CD to the computer.

Browse to the Windows folder and run the autorun file

Press next... A lot.
